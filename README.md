### shanbei-web

基于React生态开发的扇贝web 单词学习仿制项目,**本项目仅用于学习研究**.

#### 本项目使用的技术/包/框架:

前端: react(ui框架); typescript(类型约束,代码更健壮安全); redux(全局状态); react-router(路由); tailwindcss(css框架);axios(api库); ionicons(图标库).

后端: nestjs(基于nodejs的后端框架); prisma(orm); pgsql(数据库).

#### 目前已有的功能:

- [x] 用户登录注册,包括(输入验证)
- [x] 根据类别选择书籍学习
- [x] 学习新单词,复习老单词,并记录相关学习进度
- [x] 查找单词
- [x] 打卡功能
